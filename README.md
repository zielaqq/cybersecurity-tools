# 🔐 Cybersecurity Tools

Zestaw prostych narzędzi do analizy bezpieczeństwa napisanych w Python.

## 📋 Projekty

### 1. Password Strength Checker
Sprawdza siłę hasła i podaje rekomendacje.

**Funkcje:**
- Analiza długości hasła
- Sprawdzanie złożoności (wielkie/małe litery, cyfry, znaki specjalne)
- Detekcja popularnych haseł
- Kolorowe wyniki w terminalu

**Użycie:**
```bash
python password_checker.py
